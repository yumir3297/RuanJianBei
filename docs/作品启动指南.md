# 作品启动指南（报告 & 视频组专用）

> 这份文档是给负责写报告、做 PPT、录视频的组员的。不需要懂编程，按照步骤操作就能把作品跑起来。

---

## 一、你需要准备什么

你的电脑上已经有这些了，不需要额外安装：

- ✅ Python 3.13（已装好）
- ✅ Node.js v24（已装好）
- ✅ 项目代码在 `D:\桌面\软件杯`
- ✅ 所有的 AI 模型、数据库、索引都已经就绪

**不需要联网**，模型和资料都在 D 盘本地。

---

## 二、怎么启动作品（2 分钟）

### 第一步：启动后端

> 后端 = 系统的"大脑"，负责接收问题、检索资料、生成回答

1. 打开 **PowerShell**（在开始菜单搜索"PowerShell"）
2. 复制下面这行，右键粘贴，回车：

```powershell
Set-Location "D:\桌面\软件杯\backend"; .\.venv\Scripts\python.exe -m uvicorn main:app --host 0.0.0.0 --port 8000
```

3. 看到类似下面这行就说明启动成功了：

```text
Uvicorn running on http://0.0.0.0:8000
```

**这个窗口不要关**，保持开着。

---

### 第二步：打开前端

> 前端 = 你看到的网页界面，游客点按钮、打字、看回答都在这里

1. **再开一个** PowerShell 窗口（不要关掉刚才那个）
2. 复制下面两行，依次粘贴回车：

```powershell
Set-Location "D:\桌面\软件杯\frontend"
npm run dev
```

3. 看到类似下面这行就说明启动成功了：

```text
Local: http://localhost:5173/
```

4. **打开浏览器**，在地址栏输入 `http://localhost:5173/`，回车。

**你现在应该看到"游客交互端"的页面了。**

---

## 三、怎么演示（约 5 分钟，照着操作即可）

### 操作 1：基础问答

1. 在页面底部的输入框里输入：**灵山大佛有多高**
2. 点击"发送问题"
3. 观察：回答会逐字流式蹦出来，右侧"检索来源"会显示答案来自哪里

### 操作 2：主动选择景点

1. 点击页面上方 **[01 景点讲解]** 按钮
2. 在下拉框里选一个景点，比如 **灵山大佛**
3. 再点击一个话题按钮，比如 **历史文化**
4. 在输入框输入：**它有什么特色**
5. 点击发送
6. 观察：面包屑显示"景点讲解 / 灵山大佛 / 历史文化"，回答自动聚焦灵山大佛

### 操作 3：冲突纠偏（选 A 景点但打字问 B 景点）

1. 保持选择"灵山大佛"
2. 在输入框输入：**五印坛城有什么特色**
3. 点击发送
4. 观察：面包屑自动变成"五印坛城"，系统自动纠偏

### 操作 4：快捷追问

1. 在回答下方会出现几个按钮（"继续了解历史""看看建筑特色"等）
2. 点击其中一个
3. 观察：自动发送新的追问，面包屑保持当前选择

### 操作 5：路线推荐

1. 点击页面上方 **[03 路线规划]**
2. 选择一条路线
3. 在输入框输入：**这条路线有什么景点**
4. 点击发送

### 操作 6：管理后台（给评委看完整闭环）

1. 浏览器打开 `http://localhost:8000/api/admin/overview`
2. 会看到一串数字（知识条目数、FAQ 数、问答日志数等）
3. 这是管理后台的数据概览

## 五、遇到问题怎么办

### 后端启动报错

可能是上次没关干净。关了所有 PowerShell 窗口，重试第一步。

### 前端启动报错

可能是端口被占用。关掉已打开的浏览器窗口，重试第二步。

### 浏览器显示空白

确认后端窗口和前端窗口都还在运行。刷新浏览器页面（F5）。

### 问问题没有回答

检查后端窗口有没有红色报错文字。如果有，截图发给我。
