# 布局改造样式补充说明

由于ChatView.vue文件较大，以下是需要手动添加的关键CSS样式：

## 在 `<style scoped>` 末尾添加以下样式：

```css
/* ===== 新布局：数字人中心化 ===== */
.chat-layout-new {
  display: grid;
  grid-template-columns: 40% 60%;
  gap: 0;
  min-height: calc(100vh - 140px);
  position: relative;
}

.guide-sidebar {
  position: sticky;
  top: 0;
  height: calc(100vh - 140px);
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  gap: 32px;
  padding: 40px 24px;
  background: linear-gradient(135deg, var(--lingshan-green-light) 0%, var(--lingshan-paper) 100%);
  border-right: 2px solid var(--lingshan-line);
}

.guide-avatar {
  flex-shrink: 0;
}

.guide-info {
  text-align: center;
  max-width: 400px;
}

.guide-name {
  margin: 12px 0 8px;
  color: var(--lingshan-ink);
  font-family: "STKaiti", "KaiTi", serif;
  font-size: 24px;
  font-weight: 600;
}

.guide-intro {
  margin: 0;
  color: var(--lingshan-stone);
  line-height: 1.7;
}

.conversation-main {
  display: flex;
  flex-direction: column;
  padding: 24px;
  background: var(--lingshan-paper);
}

.conversation-header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  margin-bottom: 20px;
  padding-bottom: 16px;
  border-bottom: 1px solid var(--lingshan-line);
}

/* ===== 资料来源抽屉 ===== */
.source-drawer {
  position: fixed;
  bottom: 0;
  right: 0;
  width: 60%;
  background: white;
  border-top: 2px solid var(--lingshan-gold);
  box-shadow: 0 -4px 20px rgba(0,0,0,0.1);
  transition: height 0.3s ease;
  height: 56px;
  overflow: hidden;
  z-index: 100;
}

.source-drawer.expanded {
  height: 400px;
}

.source-drawer-toggle {
  width: 100%;
  padding: 16px 24px;
  border: none;
  background: linear-gradient(135deg, var(--lingshan-gold-light), white);
  color: var(--lingshan-gold-deep);
  font-size: 15px;
  font-weight: 600;
  text-align: left;
  cursor: pointer;
  display: flex;
  align-items: center;
  gap: 12px;
  transition: background 0.2s ease;
}

.source-drawer-toggle:hover {
  background: var(--lingshan-gold-light);
}

.source-drawer-toggle span:first-child {
  font-size: 12px;
  transition: transform 0.3s ease;
}

.source-drawer-content {
  height: calc(100% - 56px);
  overflow-y: auto;
  padding: 20px 24px;
}

/* ===== 响应式适配 ===== */
@media (max-width: 1200px) {
  .chat-layout-new {
    grid-template-columns: 35% 65%;
  }
  
  .guide-sidebar {
    padding: 32px 16px;
  }
}

@media (max-width: 900px) {
  .chat-layout-new {
    grid-template-columns: 1fr;
    grid-template-rows: 320px 1fr;
  }
  
  .guide-sidebar {
    position: relative;
    height: 320px;
    border-right: none;
    border-bottom: 2px solid var(--lingshan-line);
  }
  
  .source-drawer {
    width: 100%;
  }
}

@media (max-width: 560px) {
  .conversation-main {
    padding: 16px;
  }
  
  .guide-sidebar {
    height: 280px;
    padding: 20px 12px;
  }
}
```

## 样式添加位置

在 ChatView.vue 的 `</style>` 标签之前添加上述CSS。

## 验证要点

1. 左侧数字人区域占40%，固定sticky定位
2. 右侧对话区域占60%
3. 底部资料来源抽屉默认收起（56px高），点击展开（400px高）
4. 移动端（<900px）改为上下布局
