# Live2D 资源下载指南

## 需要下载的资源

### 1. Live2D Cubism SDK for Web

**下载地址：** https://www.live2d.com/download/cubism-sdk/

**步骤：**
1. 访问上述网址
2. 找到 "Cubism SDK for Web" 并点击下载
3. 解压下载的压缩包
4. 找到文件：`Core/live2dcubismcore.min.js`
5. 复制到：`frontend/public/live2dcubismcore.min.js`

### 2. Live2D 示例模型（Haru）

**下载地址：** https://www.live2d.com/download/sample-data/

**步骤：**
1. 访问上述网址
2. 下载 "Haru" 或 "HiYori" 示例模型
3. 解压下载的压缩包
4. 将整个模型文件夹复制到：`frontend/public/models/haru/`
5. 确认存在文件：`frontend/public/models/haru/haru.model3.json`
