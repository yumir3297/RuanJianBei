# 2026-06-16 阶段 B2.3 小样本真实评测执行计划

## 执行目标

在 B2.2 已完成 100 题离线评测集的基础上，先抽取前 5 题进行真实端到端评测，验证：

1. `run_e2e_chat_eval.py` 能正确请求本地后端 SSE。
2. 后端能进入真实 DeepSeek 文本 LLM 分支。
3. 评测报告能记录准确率、证据引用率、源命中率、拒答合规率和延迟。
4. 冷启动题与热运行题能分开统计。

## 本轮范围

- 使用测试集：`eval/testset/e2e_qa_100.json`
- 执行模式：`sampled-real`
- 题量：`5`
- 目标端口：临时后端 `127.0.0.1:8010`
- 报告输出：`eval/reports/e2e_eval_b2_sampled_real.json`

## 本轮不做

1. 不跑 100 题批量真实评测。
2. 不接入图片识别或视觉模型。
3. 不修改模型供应商。
4. 不删除历史数据库缓存。
5. 不长期保留临时后端进程，评测后关闭。

## 风险控制

1. 若后端无法启动，停止并记录原因。
2. 若 5 题中出现脚本级错误，先修脚本，不继续扩大题量。
3. 若模型调用异常或延迟过高，记录到报告，不直接进入 100 题。
4. 真实调用只限 5 题，后续批量评测必须再次确认。
