## 三个改动

### A. 置信度→情感映射（后端 pipeline.py，+30 行）

**当前问题**：不管什么回答，数字人永远 `emotion: "neutral"`。FAQ 答对了也面无表情，盲区拒答也不道歉。

**要做的**：在 `pipeline.py` 的 `_emit_final_answer` 出口处，根据回答路径决定 avatar emotion。

```python
# pipeline.py _emit_final_answer 中
def _answer_emotion(hit_level: str, source_count: int) -> str:
    if hit_level is None and source_count == 0:
        return "apology"
    if hit_level and hit_level.startswith("faq_"):
        return "happy"
    if hit_level == "rag_evidence" and source_count >= 2:
        return "happy"
    if hit_level == "rag_evidence":
        return "speaking"
    if hit_level == "stub" or hit_level == "fallback":
        return "apology"
    return "speaking"
```

在生成 avatar StreamEvent 时使用：
```python
emotion = _answer_emotion(hit_level, len(sources))
yield StreamEvent(type="avatar", data={"viseme_text": "", "emotion": emotion})
```

**不动 SSE 协议**，前端 `useAvatar.js` 的 `handleAvatarEvent` 已经能处理 `happy`/`speaking`/`apology`。

---

### B. 讲解模式选择器（前端 ChatView.vue，+20 行）

**当前问题**：所有游客得到同一套回答。

**要做的**：在聊天输入框上面加一个 `el-radio-group`，让游客选"儿童模式 / 标准模式 / 专业模式"，选完后在发送的消息前面拼简短指令。

```html
<!-- composer 上方 -->
<div class="explanation-mode">
  <el-radio-group v-model="explanationLevel" size="small">
    <el-radio-button value="child">儿童</el-radio-button>
    <el-radio-button value="adult">标准</el-radio-button>
    <el-radio-button value="expert">专业</el-radio-button>
  </el-radio-group>
</div>
```

```js
// submitQuery 前拼接
const levelPrefix = {
  child: '请用小朋友能听懂的语言解释：',
  adult: '',
  expert: '请给出专业详细的解释：',
}[explanationLevel.value];
submitQuery(levelPrefix + query.value);
```

**不碰后端**，不改系统提示词，都是 DeepSeek 天然能理解的指令。缓存不会污染——加了前缀后 query 变了，缓存键自然不同。

### C. 拟人化图像响应（前端 ChatView.vue，+5 行）

**当前问题**：图片识别后展示的是冷冰冰的 JSON（候选景点、视觉标签、置信度）。

**要做的**：在 vision-result-card 的候选景点标签后面加一行拟人化问候。

```html
<p v-if="visionResult?.candidate_attractions?.length" class="vision-greeting">
  我看到这是{{ visionResult.candidate_attractions[0] }}，让我为你介绍它！
</p>
```

```css
.vision-greeting {
  color: #0f766e;
  font-weight: 500;
  margin: 8px 0;
}
```

**不改后端**。Qwen 识别到灵山大佛 → 页面显示"我看到这是灵山大佛，让我为你介绍它！"
