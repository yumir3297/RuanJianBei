## 🎬 演示注意事项

### 演示前准备
1. **预热系统**：启动后端后先问一个问题，触发模型加载（避免首次冷启动）
2. **检查环境变量**：`.env` 中 DeepSeek + Qwen API key 有效
3. **确认音频**：电脑音量开启，数字人语音可播放
4. **准备图片**：提前下载一张灵山大佛清晰图片（用于识景演示）
5. **清空缓存**：清除浏览器缓存，确保干净演示环境
6. **备用网络**：如果现场网络不稳定，FAQ+选择功能仍可演示（不依赖大模型）

### 演示技巧
1. **语速控制**：讲解词要清晰简洁，避免啰嗦
2. **等待时机**：提问后等待2-3秒让数字人完成状态切换和回答
3. **突出对比**：标准模式 vs 儿童模式的对比要明显
4. **证据展示**：点击右侧证据卡片，展示可追溯性
5. **表情观察**：提醒评委注意数字人表情随问答质量变化

### 时间调整
