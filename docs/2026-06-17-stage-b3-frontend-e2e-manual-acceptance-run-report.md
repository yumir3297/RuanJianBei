## 运行说明

## 人工验收步骤

1. 打开 `http://127.0.0.1:5173/`。
2. 在“拍照识景”区域点击“选择图片”。
3. 选择测试图片，例如：

```text
D:\桌面\软件杯测试图片\BHIBGIFHFFGBA-mZwPOX6MrF.png
```

4. 点击“识别图片”。
5. 等待 Qwen 识别结果展示候选景点、视觉标签、置信度和检索线索。
6. 点击“根据图片线索提问”。
7. 检查聊天回答是否包含 `[证据N]`。
8. 检查右侧“检索来源”是否展示官方资料来源。

## 注意事项

1. 图片识别会真实调用 Qwen，可能需要 20 秒以上。
2. 一键提问可能触发后端问答链路。
3. 图片识别结果只作为检索线索，最终事实仍以官方资料证据为准。
4. 如果测试完成后需要关闭服务，让 Codex 停止本轮启动的本地服务即可。
