## 创意说明（随视频提交的独立文档）

### 一、产品创意来源
- 景区导览痛点：人工导游供不应求、固定语音导览机交互单一、景区管理缺乏数据支撑
- 灵山胜境独特需求：佛教文化景点，讲解需要准确的历史文化知识支撑
- 现有竞品不足：语音导览机无法回答自由提问，人工讲解质量参差不齐

### 二、核心创意理念
- 让 AI 有据可依：RAG 证据链，所有回答基于官方资料，杜绝大模型幻觉
- 让交互有温度：数字人表情联动，六种状态自然切换
- 让知识可生长：盲区自动聚合、人工补录、索引自刷新完整闭环
- 让系统可信赖：100 题端到端评测，全链路可量化可回归

### 三、创新亮点
1. 五级检索通道（FAQ L1/L2/L3 + 缓存 + Chroma RAG）+ 知识盲区自动闭环
2. GPS 弱信号混合定位方案（GPS → WiFi指纹 → 手动选择三级降级）
3. 情感分析驱动的数字人表情联动 + 后台感受度报告

### 四、技术独特性
- 条件式 Reranker：通过参数矩阵评测确定跳过条件，省时 86% 不降准确率
- FAQ L3 实体+意图双门控：100% 正例准确率 + 100% 盲区拒绝率
- 真实多模态大模型参与业务：DeepSeek 文本 + Qwen 视觉，非仅文本接口
- 全链路可评测：100 题固定评测集，十项指标定量追踪
