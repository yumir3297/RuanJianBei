## 一、执行说明

运行评测脚本：

```powershell
# 1. 先启动后端
Set-Location "D:\桌面\软件杯\backend"
.\.venv\Scripts\python.exe -m uvicorn main:app --host 0.0.0.0 --port 8000

# 2. 另开终端，运行评测
Set-Location "D:\桌面\软件杯"
.\backend\.venv\Scripts\python.exe evaluation\evaluate.py
```

评测脚本 `evaluation/evaluate.py` 将：
- 逐题调用 `/api/chat/stream` SSE 接口
- 自动评分（1-5分）
- 统计分类准确率、延迟 P50/P95、来源展示率
- 输出 `evaluation/eval_report_<timestamp>.json` 详细报告
