## 三、项目当前结构与主要功能

### 3.1 架构总览

```
前端 (Vue 3 + Vite)
  ├── /                     LandingPage（入口选择 + 景区轮播）
  ├── /tourist/welcome       HomePage（开始游览 + 数字人展示）
  ├── /tourist/select        ModeSelectView（讲解方式：亲子/休闲/深度/自由）
  ├── /tourist               ChatView（数字人讲解主页面 + 左右操作面板）
  ├── /admin/login           AdminLogin（前端密码登录）
  ├── /admin/dashboard       Dashboard（数据总览 + ECharts）
  ├── /admin/knowledge       KnowledgeManage（知识CRUD）
  ├── /admin/avatar          AvatarConfig（数字人配置）
  ├── /admin/analytics       AnalyticsReport（数据大屏）
  ├── /admin/blind-spots     BlindSpotManage（知识盲区管理）
  └── /admin/assets          DisplayAssets（资源替换）

后端 (FastAPI + SQLite + ChromaDB)
  └── :8000/api/
      ├── /chat/stream       SSE流式问答（核心）
      ├── /admin/*           管理后台接口
      ├── /knowledge/*       知识管理接口
      ├── /quick-select/*    主动选择
      ├── /vision/*          图片识别
      ├── /asr/*            语音识别
      ├── /insights/*        数据洞察
      └── /voice/*           语音合成
```

## 十三、当前最值得保留和强化的三个亮点
