# 队友快速上手 — 一键启动指南

> API Key 已配好，不需要你自己申请任何账号。照下面做，15 分钟内跑起来。

---

## 你的电脑需要安装

- **Python 3.13**：[python.org](https://www.python.org/downloads/) 下载，安装时勾选 "Add Python to PATH"
- **Node.js v20+**：[nodejs.org](https://nodejs.org/) 下载 LTS 版

验证安装：
```powershell
python --version    # 应该显示 Python 3.13.x
node --version      # 应该显示 v20.x 或更高
```

---

## 第一步：拿到项目文件

从我这里拷贝整个项目文件夹到你的 D 盘（或任意位置）。

**重要：拷的时候排除这两个目录（太大了，装环境时会重新生成）：**
- `backend\.venv\`（Python 虚拟环境，500MB+）
- `frontend\node_modules\`（npm 包，600MB+）

**但务必带上这两个目录（里面有 AI 模型和向量索引）：**
- `.cache\huggingface\`（模型缓存，1.5GB，拷过去就不用重新下载）
- `kb\chroma_db\`（向量索引，拷过去就不用 rebuild）

```
你的电脑拷完后应该是这样的：
D:\软件杯\
  backend\          （没有 .venv 目录）
  frontend\         （没有 node_modules 目录）
  data\
  kb\               （有 chroma_db）
  .cache\           （有 huggingface）
  ...
```

---

## 第二步：装后端

打开 PowerShell，逐条执行：

```powershell
# 进入后端目录
cd D:\软件杯\backend

# 创建虚拟环境（只需执行一次）
python -m venv .venv

# 激活虚拟环境
.\.venv\Scripts\activate

# 安装依赖（CPU 版 PyTorch 先装）
pip install torch --index-url https://download.pytorch.org/whl/cpu
pip install -r requirements.txt
```

---

## 第三步：装前端

另开一个 PowerShell：

```powershell
cd D:\软件杯\frontend
npm install
```

---

## 第四步：初始化数据（只需第一次做）

先启动后端：

```powershell
cd D:\软件杯\backend
.\.venv\Scripts\activate
python -m uvicorn main:app --host 127.0.0.1 --port 8000
```

保持这个终端开着，另开一个终端执行：

```powershell
# 导入景区数据
curl -X POST http://127.0.0.1:8000/api/admin/sync-processed-data

# 构建向量索引（首次会下载两个 AI 模型，约 1.5GB，需联网）
curl -X POST http://127.0.0.1:8000/api/admin/build-rag-index
```

看到成功提示后，初始化完成。

---

## 第五步：日常启动

每次要跑项目时：

**终端 1 — 启动后端：**
```powershell
cd D:\软件杯\backend
.\.venv\Scripts\activate
python -m uvicorn main:app --host 127.0.0.1 --port 8000
```

**终端 2 — 启动前端：**
```powershell
cd D:\软件杯\frontend
npm run dev
```

浏览器打开：
- 游客端：`http://localhost:5173`
- 管理后台：`http://localhost:5173/admin/dashboard`

## 遇到问题？

| 症状 | 原因 | 解决 |
|------|------|------|
| `python` 命令找不到 | Python 没装或没加 PATH | 重装 Python 并勾选 "Add to PATH" |
| `pip install` 很慢 | 默认源在国外 | `pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple` |
| 前端页面空白 | 后端没启动 | 先确认 8000 端口后端在跑 |
| 问问题不回复 | DeepSeek 余额不足 | 暂时用文本降级模式 |
| Live2D 不显示 | CDN 没加载到 | 联网状态下多等几秒 |
